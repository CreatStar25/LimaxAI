## 1. Layoutgrund for omslag och affischer

- **Hjulfokuspunkt**: modell eller produkt som visuellt ankare—undvik rora.
- **Titelsaker zon**: utrymme for masthead och rubriker.
- **Seriekonsekvens**: las palett och ljusord.

## 2. Promptstruktur

Anvand **scen + redaktionell modemotiv + ljus + komposition (negativt utrymme) + kvalitet**. Exempel:

```text
Fashion magazine cover layout, editorial photography, negative space for headline, soft studio lighting, elegant pose, high-end styling, 8k, color grading cinematic
```

| Modul | Roll | Tips |
|-------|------|------|
| Negativt utrymme | Plats for rubriker | negative space, ren bakgrund |
| Belysning | Premiumkansla | softbox, kantljus, mjukt kontrast |
| Stil | Magasinslook | editorial, lyxkampanj |

## 3. Produktionsflode

1. Valj **bildformat** efter kanal (t.ex. 3:4 eller 4:5 stende).
2. Las komposition sedan farg—skriv inte om hela prompten.
3. Typografi i efterhand; AI-fasen fokuserar pa **ren hjaltedbild**.

## 4. Slutsats

Modoomslag kraver **hjaltedbild + negativt utrymme + ljus**. Pa **LimaxAI** iterera snabbt och valj basta bildruta for layout och publicering.
