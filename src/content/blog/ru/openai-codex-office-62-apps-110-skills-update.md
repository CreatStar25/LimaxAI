## Цифры: 5 млн WAU, не-разработчики растут быстрее

У Codex более **5 млн** еженедельных пользователей (~**×6** с февраля). **Не-разработчики** ~**20%**, рост ~**×3** быстрее разработчиков.

| Тип | Рост за неделю (прибл.) |
| --- | --- |
| Анализ данных | +110% |
| Исследования | +37% |
| Отчёты, мемо, контракты, слайды, таблицы | +36% |

**60%+** запускают несколько задач Codex за день (в апреле — меньше половины). Проникновение в офисную работу, не только «помощник программисту».

## Шесть плагинов: цепочки инструментов по ролям

Каждый плагин объединяет SaaS роли; вы даёте инструкцию на естественном языке — Codex оркестрирует в фоне.

| Плагин | Инструменты | Пример |
| --- | --- | --- |
| Данные | Snowflake, Tableau | Падение выручки по регионам и причины |
| Креатив | Figma, Canva | Варианты рекламы из брифа |
| Продажи | Salesforce, HubSpot | Приоритетные аккаунты, встречи, CRM |
| Дизайн продукта | Figma, Canva | Кликабельные прототипы, аудит флоу |
| Акции | FactSet, S&P, PitchBook | Отчётность, comps, проверка тезиса |
| IB | Research и due diligence | Pitch для клиента, comps |

Планируются финансы, PE, маркетинг, консалтинг, legal и партнёрская экосистема. **Слой оркестрации SaaS**.

## Sites: одна фраза — интерактивная страница

**Sites** в превью для Business и Enterprise: **веб-страница**, а не только файл.

- Финансы: Excel → сценарный планировщик в браузере
- Хаб запуска
- Страница перед ревью с клиентом

От недель разработки до минут диалога; страницы обновляются по мере изменения данных.

## Annotations: выделить и править локально

Не только код: **документы, таблицы, слайды**. Codex меняет только выделенное.

## Codex и ChatGPT: сближение

Чат — вход, выполнение workflow — продукт. **ChatGPT** и Codex на одной траектории.

## Кто почувствует давление первым

Salesforce/Snowflake держат ров за счёт данных и compliance. Давление на **средний слой с «удобным UI»**: лёгкий BI, PM, генераторы отчётов.
