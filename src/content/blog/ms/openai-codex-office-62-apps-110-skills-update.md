## Angka: 5 juta WAU, bukan dev berkembang lebih pantas

Codex melebihi **5 juta** pengguna aktif mingguan (~**6×** sejak Februari). **Bukan pembangun** ~**20%**, ~**3×** lebih pantas.

| Jenis | Pertumbuhan mingguan (anggaran) |
| --- | --- |
| Analisis data | +110% |
| Penyelidikan | +37% |
| Hasil kerja (laporan, memo, kontrak, slaid) | +36% |

**60%+** menjalankan beberapa tugas Codex sehari.

## Enam plugin: rantai alat mengikut peranan

Setiap plugin menggabungkan SaaS peranan; arahan bahasa natural, Codex mengorkestra di belakang tabir.

| Plugin | Penyambung | Contoh |
| --- | --- | --- |
| Data | Snowflake, Tableau | Penurunan hasil mengikut wilayah |
| Kreatif | Figma, Canva | Varian iklan |
| Jualan | Salesforce, HubSpot | CRM, mesyuarat |
| Reka bentuk produk | Figma, Canva | Prototaip klik |
| Saham | FactSet, S&P | Comps |
| IB | Penyelidikan | Pitch |

Lebih banyak plugin dan ekosistem rakan kongsi. **Lapisan orkestrasi SaaS**.

## Sites: satu ayat, satu halaman interaktif

**Sites** pratonton untuk Business dan Enterprise.

- Kewangan: Excel → perancang senario
- Hab pelancaran
- Halaman pra-semakan pelanggan

Dari minggu pembangunan ke minit dialog.

## Annotations: pilih dan sunting setempat

Melangkaui kod: **dokumen, hamparan, slaid**.

## Codex dan ChatGPT: penumpuan

Sembang sebagai pintu masuk, pelaksanaan aliran kerja sebagai produk.

## Siapa rasa tekanan dahulu

Salesforce/Snowflake: parit data/pematuhan. Tekanan pada **lapisan UI pertengahan**.
