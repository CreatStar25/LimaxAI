Hari ini, DeepSeek secara rasmi mengumumkan pelan harga API untuk model baharu siri V4, menandakan pelancaran komersial rasmi model besar generasi baharu yang dinanti-nantikan ini untuk pembangun di seluruh dunia. Siri DeepSeek V4 merangkumi dua versi: Flash dan Pro, kedua-duanya menyokong konteks ultra panjang 1 juta token. Sementara prestasi telah meningkat dengan ketara, pengoptimuman kos telah dicapai, menyediakan pilihan yang lebih kompetitif untuk pembangunan aplikasi AI.

## Penerangan Model dan Harga
Harga API DeepSeek dikira per juta token. Token adalah unit teks terkecil yang dikenal pasti oleh model, yang boleh menjadi perkataan, nombor atau tanda baca. Yuran dikira berdasarkan jumlah token input dan output yang diproses oleh model.

### Perbandingan Parameter Model Terperinci
| Ciri | deepseek-v4-flash* | deepseek-v4-pro |
|---------|---------------------|------------------|
| BASE URL (Format OpenAI) | https://api.deepseek.com | https://api.deepseek.com |
| BASE URL (Format Anthropic) | https://api.deepseek.com/anthropic | https://api.deepseek.com/anthropic |
| Versi Model | DeepSeek-V4-Flash | DeepSeek-V4-Pro |
| Mod Pemikiran | Menyokong kedua-dua mod bukan pemikiran dan pemikiran (lalai). Lihat Mod Pemikiran untuk arahan pertukaran | Menyokong kedua-dua mod bukan pemikiran dan pemikiran (lalai). Lihat Mod Pemikiran untuk arahan pertukaran |
| Panjang Konteks | 1M | 1M |
| Panjang Output Maksimum | Sehingga 384K | Sehingga 384K |
| Output JSON | ✓ | ✓ |
| Panggilan Alat | ✓ | ✓ |
| Penyelesaian Awalan Sembang (Beta) | ✓ | ✓ |
| Penyelesaian FIM (Beta) | Mod bukan pemikiran sahaja | Mod bukan pemikiran sahaja |
| 1M Token Input (Cache Hit) | $0.028 | $0.145 |
| 1M Token Input (Cache Miss) | $0.14 | $1.74 |
| 1M Token Output | $0.28 | $3.48 |

* Nota: Nama model `deepseek-chat` dan `deepseek-reasoner` akan dihentikan secara beransur-ansur pada masa hadapan. Untuk keserasian, kedua-dua nama ini pada masa ini masing-masing sepadan dengan mod bukan pemikiran dan mod pemikiran `deepseek-v4-flash`.

## Peraturan Potongan Yuran
Jumlah penggunaan sebenar = penggunaan token × harga unit yang sepadan. Yuran akan ditolak terus daripada baki pengecasan atau baki hadiah anda. Apabila kedua-dua baki wujud, sistem akan memberi keutamaan kepada penggunaan baki hadiah.

Harga produk mungkin diselaraskan mengikut keadaan pasaran, dan DeepSeek mengekalkan hak muktamad untuk menyesuaikan harga. Pembangun dinasihatkan untuk mengecas semula mengikut keperluan penggunaan sebenar dan menyemak secara berkala halaman harga rasmi untuk maklumat terkini.

## Analisis Keunggulan Utama
Sambil mengekalkan prestasi tinggi, siri DeepSeek V4 melaksanakan strategi harga yang sangat kompetitif, yang amat sesuai untuk pelbagai senario aplikasi AI:

### Pilihan Kuasa Pengiraan Kos Efektif
Versi deepseek-v4-flash menyediakan prestasi cemerlang pada harga yang sangat rendah, terutamanya sesuai untuk senario yang memerlukan kelajuan respons tinggi dan keluaran tinggi, seperti khidmat pelanggan pintar, penjanaan kandungan, aplikasi sembang biasa, dan lain-lain. Harganya hanya $0.028 per juta token input apabila cache kena dengan ketara mengurangkan kos operasi aplikasi berskala besar.

### Pengalaman Prestasi Perdana
Versi deepseek-v4-pro dioptimumkan untuk tugas penaakulan kompleks dan bidang profesional, dan menunjukkan prestasi lebih baik dalam senario seperti pengiraan matematik, penjanaan kod, dan soal jawab pengetahuan profesional, sesuai untuk senario aplikasi profesional yang memerlukan output berketepatan tinggi.

### Sokongan Konteks Ultra Panjang
Kedua-dua versi menyokong konteks ultra panjang 1 juta token dan panjang output maksimum 384K, yang boleh mengendalikan tugas teks sangat besar seperti keseluruhan buku, dokumen panjang, dan kod projek lengkap, menyediakan asas yang kukuh untuk membangunkan aplikasi AI yang kompleks.
