## İpucu nereden: Codex günlüklerinde canary

GPT-5.5'ten yaklaşık üç hafta sonra **Codex yönlendirmesi** kısa süre `gpt-5.6` gösterdi sonra kayboldu.

**OpenAI GPT-5.6 duyurmadı.**

## Dahili kod adları: iris-alpha, ember-alpha, beacon-alpha

**iris-alpha**, **ember-alpha**, **beacon-alpha** geçiyor; genel katman eşlemesi belirsiz.

| Kod | Kamuya açık bilgi | Not |
| --- | --- | --- |
| iris-alpha | GPT-5.6 adıyla | Sürüme en yakın |
| ember-alpha | Günlükte | Rol belirsiz |
| beacon-alpha | Günlükte | Rol belirsiz |

## 1,5M bağlam: GPT-5.5'e göre

Sızıntılar GPT-5.6 için yaklaşık **1.500.000 token** diyor:

| Kanal | Yaklaşık üst sınır | 1,5M'ye göre |
| --- | --- | --- |
| GPT-5.5 API | 1.050.000 | ~+43% |
| GPT-5.5 (Codex OAuth) | 400.000 | ~+275% |
| GPT-5.6 (sızıntı) | 1.500.000 | — |

Daha az manuel parçalama, daha yüksek maliyet ve gecikme.

## Aşırı uzunluk stres testleri (resmi değil)

**OpenCode** vb.: ~**900k token** yanıt veriyor; >**1,05M** örnekler var. SLA değil.

## UI üretimi

Sızıntıda kısa istemle **Lumen Notes**—daha düzenli ızgara ve tipografi.

## Hâlâ belirsiz

Parametreler, API fiyatı, ChatGPT planları—resmi duyuru beklenmeli.

## Yayın öncesi GPT eğitimi

GPT-5.6 gelmeden önce **GPT-5.5 / GPT-5.4** ile alışkanlık edinin:

1. **Uzun girdiyi sıkıştırın**—ilgisiz bölümleri atın, özet veya içindekiler kullanın.
2. **Durma kuralları yazın**—ör. «en fazla iki arama turu», «kanıt yeterliyse dur».
3. **UI’yi katmanlayın**—önce bilgi mimarisi ve bileşen listesi, sonra kod.
4. **Sabit eval seti**—aynı soruları `low` / `medium` ile çalıştırın, gecikme/maliyet kaydedin.
