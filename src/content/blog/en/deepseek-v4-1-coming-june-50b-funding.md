## 1. Massive Funding: The Happy Problem of Capacity Expansion

DeepSeek has secured a new funding round of 50 billion yuan (approximately $6.9 billion USD), with its valuation rising from $20 billion to $45 billion. Although DeepSeek is not a cash-strapped institution, this funding round is primarily aimed at acquiring more computing resources.

After the API price reduction at the end of April, DeepSeek saw a massive surge in API usage, causing several service interruptions due to computing resource constraints.

DeepSeek V4.1 is expected to be released in June. The upgrade is anticipated to focus on the following capabilities:

- **Improved coding capabilities**: Coding is currently the biggest weakness of DeepSeek V4 compared to Claude Opus 4.6/4.7 and GPT-5.4/5.5. If V4.1 can close this gap, it will truly be able to compete with top foreign models.

- **Enhanced multimodal capabilities**: DeepSeek has been conducting grayscale testing of visual capabilities, but user feedback has been mixed. The new release may bring improvements in this area.

- **API stability**: With the explosive growth in users, service stability has become an urgent priority. A significant portion of the funding will go toward solving this issue.



## 2. What to Expect from DeepSeek V4.1?

DeepSeek V4.1 is widely seen as the key version to address the above shortcomings. If programming and multimodal capabilities can catch up with the top overseas tier, and API stability is fundamentally improved, DeepSeek's position in the global large model competition will be further consolidated. The computing power gained from the 50 billion financing is being transformed into tangible product experience improvements.

## 3. Market Position and Impact on Users

For ordinary users, DeepSeek's rapid development means one thing: better, more stable, and cheaper AI services are on the way. If you're already using DeepSeek V4, the V4.1 in June is worth looking forward to; if you haven't tried it yet, now is the best time.
