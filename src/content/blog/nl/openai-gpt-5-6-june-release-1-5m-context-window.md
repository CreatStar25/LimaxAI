## Waar de aanwijzing vandaan komt

~3 weken na GPT-5.5 verscheen kort een **routingregel** naar `gpt-5.6` in Codex-logs.

**OpenAI heeft GPT-5.6 niet aangekondigd.**

## 1,5M context vs GPT-5.5

Leaks: ~**1.500.000 tokens**.

| Kanaal | Limiet (ca.) | vs 1,5M |
| --- | --- | --- |
| GPT-5.5 API | 1.050.000 | ~+43% |
| GPT-5.5 (Codex OAuth) | 400.000 | ~+275% |
| GPT-5.6 (lek) | 1.500.000 | — |

Minder handmatig chunken, hogere kosten en latentie.

## Stresstests (onofficieel)

~**900k tokens** werkt nog; soms >**1,05M** — geen SLA.

## Nog onduidelijk

Parameters, API-prijzen, ChatGPT-plannen — wacht op aankondiging.

## GPT-tutorial vóór launch

Voor GPT-5.6 landt, oefen op **GPT-5.5 / GPT-5.4**:

1. **Langere input inkorten**—irrelevante delen weg, inhoudsopgave of korte samenvatting.
2. **Stopregels** voor agents/code—bijv. «maximaal twee zoekrondes».
3. **UI in lagen**—eerst informatiearchitectuur en componentenlijst, dan code.
4. **Vaste eval-set**—dezelfde prompts op `low` / `medium`, log latency en kosten.
