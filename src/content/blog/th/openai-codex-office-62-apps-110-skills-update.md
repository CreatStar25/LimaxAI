## ตัวเลข: WAU 5 ล้าน ผู้ที่ไม่ใช่ dev โตเร็วกว่า

Codex มีผู้ใช้รายสัปดาห์ **5 ล้าน** (~**6×** ตั้งแต่ก.พ.) **ไม่ใช่นักพัฒนา** ~**20%** โต ~**3×** เร็วกว่า

| ประเภท | โตรายสัปดาห์ (โดยประมาณ) |
| --- | --- |
| วิเคราะห์ข้อมูล | +110% |
| วิจัย | +37% |
| ผลงานความรู้ | +36% |

**60%+** รันหลายงาน Codex ต่อวัน

## หกปลั๊กอิน: toolchain ตามบทบาท

แต่ละปลั๊กอินรวม SaaS ของบทบาท สั่งภาษาธรรมชาติ Codex ประสานเบื้องหลัง

| ปลั๊กอิน | เชื่อมต่อ | ตัวอย่าง |
| --- | --- | --- |
| ข้อมูล | Snowflake, Tableau | รายได้ลดตามภูมิภาค |
| ครีเอทีฟ | Figma, Canva | โฆษณาหลายแบบ |
| ขาย | Salesforce, HubSpot | CRM ประชุม |
| ออกแบบผลิตภัณฑ์ | Figma, Canva | โปรโตไทป์คลิกได้ |
| หุ้น | FactSet, S&P | Comps |
| IB | วิจัย | Pitch |

ปลั๊กอินและพันธมิตรเพิ่ม **ชั้นประสาน SaaS**

## Sites: หนึ่งประโยค หนึ่งหน้าโต้ตอบ

**Sites** พรีวิว Business/Enterprise

- การเงิน: Excel → วางแผนสถานการณ์บนเบราว์เซอร์
- ฮับเปิดตัว
- หน้าก่อนรีวิวลูกค้า

จากสัปดาห์พัฒนาเป็นคุยนาที

## Annotations: เลือกแล้วแก้เฉพาะจุด

นอกโค้ด: **เอกสาร สเปรดชีต สไลด์** แก้เฉพาะที่เลือก

## Codex กับ ChatGPT: บรรจบ
