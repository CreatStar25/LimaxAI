## 1. Essentiel couverture et affiche

- **Point focal** : mannequin ou produit comme ancre ; evitez le fouillis.
- **Zone titre** : reservez l'espace pour le masthead et les titres.
- **Serie** : verrouillez palette et mots d'eclairage.

## 2. Structure du prompt

Utilisez **scene + photo mode + lumiere + composition (espace negatif) + qualite**. Exemple :

```text
Fashion magazine cover layout, editorial photography, negative space for headline, soft studio lighting, elegant pose, high-end styling, 8k, color grading cinematic
```

| Module | Role | Astuce |
|--------|------|--------|
| Espace negatif | Place pour les titres | negative space, fond propre |
| Lumiere | Rendu premium | softbox, contre-jour, contraste doux |
| Style | Look magazine | editorial, campagne luxe |

## 3. Production

1. Choisissez le **ratio** selon le canal (ex. 3:4 ou 4:5 vertical).
2. Fixez la composition puis la couleur ; evitez de tout reecrire.
3. Typographie en post ; l'IA fournit un **hero visuel propre**.

## 4. Conclusion

Les couvertures mode demandent **hero + espace negatif + lumiere**. Sur **LimaxAI**, iterez vite et choisissez la meilleure image pour la mise en page.
