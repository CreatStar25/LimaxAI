## 先看數據：500 萬週活，非開發者增速更快

Codex 週活躍用戶已突破 **500 萬**，自 2 月桌面版上線以來約 **6 倍** 成長。更值得看的是結構：**非開發者**（分析師、行銷、營運、設計師、研究員、投資人等）已占約 **20%**，增速約為開發者的 **3 倍以上**。

| 任務類型 | 近期週環比（約） |
| --- | --- |
| 數據分析 | +110% |
| 研究類 | +37% |
| 知識產出（報告、備忘錄、合約、PPT、表格等） | +36% |

另有行為數據：**60% 以上** 用戶會在一天內並行跑多個 Codex 任務（4 月中旬還不到一半）。這說明 Codex 正在白領日常工作中滲透，而不只是「程式設計師多了一個助手」。

## 六個外掛：按角色打包職業工具鏈

設計思路是按崗位定制，而不是把通用能力硬套給所有人。每個外掛打包該角色常用軟體，你用自然語言下指令，Codex 在後台跨應用跑完整流程。

| 外掛 | 連接的典型工具 | 能做什麼（示例） |
| --- | --- | --- |
| 數據分析 | Snowflake、Tableau 等 | 問「上季度哪個區域收入下降、原因是什麼」，查庫、分析、出圖一氣呵成 |
| 創意製作 | Figma、Canva 等 | 根據創意簡報生成廣告素材變體、產品場景圖、電商用圖 |
| 銷售 | Salesforce、HubSpot 等 | 找高優先級客戶、準備會議材料、跟進郵件、更新 CRM、做關單與風險復盤 |
| 產品設計 | Figma、Canva 等 | 線框圖轉可點擊原型，或從網址審計用戶流程 |
| 公募股權投資 | FactSet、S&P、PitchBook 等 | 盈利分析、公司對比、投資邏輯驗證 |
| 投行 | 研究與盡調材料 | 生成客戶就緒 Pitch、可比公司與可比交易分析 |

後續還有企業財務、私募、行銷策略、諮詢、法律等外掛，並開放生態讓合作夥伴上架。OpenAI 在這裡扮演 **跨 SaaS 的調度層**：不替代某一個工具，而是坐在工具之上串聯流程。

## Sites：一句話生成可分享的互動網頁

**Sites** 面向 Business / Enterprise 用戶預覽開放。以前 Codex 幹完活往往給你一個檔案；現在可以直接生成 **網頁**，把連結發給同事即可查看和操作。

典型場景包括：

- 財務把 Excel 模型變成線上情景規劃器，管理層在網頁上調參對比假設。
- 產品發布時匯總成 Hub：最新文案、里程碑、負責人、決策記錄集中展示。
- 客戶評審前生成互動頁：產品更新、待解決問題、使用趨勢與下一步行動。

從「想要一個互動應用」到「真的有一個」，過去可能要幾週開發；現在可以壓縮到幾分鐘對話。頁面也不是一次性的——資訊變了可以讓 Codex 更新，也可用於專案進度、客服指引、團隊創意簡報等。

## Annotations：圈哪改哪

**Annotations** 從程式設計師改程式擴展到 **文件、表格、投影片**。選中不滿意的一塊，說明要怎麼改，Codex 只動選中區域，其餘內容保持不變。

例如：投資分析裡圈出一段問數據來源；投影片裡圈住圖表改標籤；Sites 頁面裡選中導覽列換字體。它解決的是「初稿之後怎麼精修」，不必整篇重來。

## Codex 與 ChatGPT：走向合併

這次三項功能疊加，信號很明確：Codex 要服務所有上班的人，而不只是工程師。OpenAI 內部 Codex 與 **ChatGPT** 已由同一團隊推進，長期看兩者很可能 **合併為一條產品線**——聊天是入口，執行工作流才是核心。

## 誰會更受影響

數據與合規壁壘深的廠商（如 Salesforce、Snowflake）短期內仍難被外掛完全替代。更需要警惕的是 **靠「介面好用」吃飯的中間層**：BI 看板、輕量專案管理、報告生成器等。當用戶只需說清楚要什麼，而不必學三天 Tableau，「介面體驗」這條護城河會變薄。壁壘會向兩頭集中：**數據與合規源頭**，或 **能理解意圖、調度工具的 Agent 層**。

## GPT 教程：在 Codex 辦公流之外，先把 ChatGPT 用熟

Codex 負責跨應用幹活；日常問答、起草文件、腦力激盪仍離不開 **ChatGPT**。在 **GPT-5.6** 正式全面開放前，建議先用 **GPT-5.4 / GPT-5.5** 練這幾件事：

1. **任務寫清邊界**：說明輸入來源、輸出格式、不要改動哪些部分（和 Annotations 的圈選邏輯一致）。
2. **長文件先給結構**：目錄 + 每節一句話目標，再讓模型展開，減少跑偏。
3. **辦公場景拆步**：先列「要連哪些系統、產出什麼檔案」，再執行，方便以後遷移到 Codex 外掛。
4. **固定樣本做對比**：同一組提示在不同推理檔位各跑一遍，記錄品質與耗時。
