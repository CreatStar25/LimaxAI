## Angka: 5 juta WAU, non-dev tumbuh lebih cepat

Codex melampaui **5 juta** pengguna aktif mingguan (~**6×** sejak Februari). **Non-developer** ~**20%**, tumbuh ~**3×** lebih cepat.

| Jenis | Pertumbuhan mingguan (kira-kira) |
| --- | --- |
| Analisis data | +110% |
| Riset | +37% |
| Output (laporan, memo, kontrak, slide, sheet) | +36% |

**60%+** menjalankan beberapa tugas Codex dalam sehari.

## Enam plugin: toolchain per peran

Setiap plugin menggabungkan SaaS peran; instruksi bahasa natural, Codex mengorkestrasi di belakang layar.

| Plugin | Konektor | Contoh |
| --- | --- | --- |
| Data | Snowflake, Tableau | Penurunan pendapatan per wilayah |
| Kreatif | Figma, Canva | Varian iklan dari brief |
| Penjualan | Salesforce, HubSpot | CRM, rapat |
| Desain produk | Figma, Canva | Prototipe klik |
| Saham | FactSet, S&P | Comps |
| IB | Riset | Pitch |

Lebih banyak plugin dan ekosistem mitra. **Lapisan orkestrasi SaaS**.

## Sites: satu kalimat, satu halaman interaktif

**Sites** pratinjau untuk Business dan Enterprise.

- Keuangan: Excel → perencana skenario di browser
- Hub peluncuran
- Halaman pra-review klien

Dari minggu pengembangan ke menit dialog.

## Annotations: pilih dan edit lokal

Di luar kode: **dokumen, spreadsheet, slide**. Hanya bagian terpilih yang berubah.

## Codex dan ChatGPT: konvergensi

Chat sebagai pintu masuk, eksekusi workflow sebagai produk.

## Siapa yang merasakan tekanan dulu

Salesforce/Snowflake: parit data/compliance. Tekanan pada **lapisan UI tengah**.
