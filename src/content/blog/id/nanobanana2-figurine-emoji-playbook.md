## Konten berbasis karakter: konsistensi dulu

Pada figur dan set emoji, identitas yang stabil lebih penting dari satu gambar sempurna: wajah, pakaian, dan bahasa pose harus selaras.

![Contoh figur dan emoji Nano Banana](https://img.limaxai.com/aiweb/limaxai-blog-5.png)

## Metode prompt
1. Kunci DNA karakter
2. Kembangkan ekspresi dan pose
3. Ekspor rasio per saluran

```text
Cute collectible figurine style character, keep same character identity, consistent face and outfit, multiple expressions set, sticker-ready transparent background, clean lighting, high detail, social media friendly
```
