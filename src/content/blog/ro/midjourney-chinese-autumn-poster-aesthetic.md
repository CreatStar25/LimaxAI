## 1. Elemente vizuale toamna stil chinezesc

- **Culoare**: rosu arțar, auriu ginkgo, saturatie tinuta in munti cerneala.
- **Motive**: varfuri indepartate, gast sălbatice, colturi pavilion—sugestie fara supraîncarcare.
- **Spatiu negativ**: rezerva pentru titluri si layout tip sigiliu.

## 2. Structura promptului

**Subiect + stil pictura/foto chinez + indicii toamna + lumina + spatiu negativ + calitate**. Exemplu:

```text
Chinese style autumn poster, ink wash and golden ginkgo, misty mountains, elegant negative space for typography, soft diffused light, fine art poster layout, ultra detailed --ar 2:3
```

| Modul | Nota |
|-------|------|
| Fuziune stil | textura cerneala, ilustratie chineza contemporana |
| Toamna | rosu arțar, frunze aurii, rau cețos |
| Layout | poster layout, zona titlu sigura |

## 3. Flux poster

1. **Schițati hero-ul** si confirmati paleta de toamna si albe.
2. Rafinati **detaliu si claritate**, tipografie dupa.
3. O paleta si set pensula fix per serie.

## 4. Concluzie

Posterele toamna chinezesti se bazeaza pe **motiv + spatiu negativ + ton unitar**. Pe **LimaxAI** cu **Midjourney**, generati variante si alegeti cea mai buna pentru layout final.
