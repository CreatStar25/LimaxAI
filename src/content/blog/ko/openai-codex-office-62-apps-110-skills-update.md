## 숫자로 보기: 주 500만 WAU, 비개발자 성장이 더 빠름

Codex 주간 활성 **500만** 돌파, 2월 데스크톱 이후 약 **6배**. **비개발자** 약 **20%**, 개발자 대비 **3배+** 성장.

| 업무 | 최근 주간 증가(대략) |
| --- | --- |
| 데이터 분석 | +110% |
| 리서치 | +37% |
| 지식 산출물(보고서, 메모, 계약, PPT, 시트) | +36% |

하루에 여러 Codex 작업을 돌리는 비율 **60%+** (4월 중순엔 절반 미만).

## 6개 플러그인: 직무별 툴체인

직무별 SaaS를 묶고 자연어 한 줄로 백그라운드에서 크로스앱 실행.

| 플러그인 | 연결 예 | 예시 |
| --- | --- | --- |
| 데이터 | Snowflake, Tableau | 지역별 매출 하락 원인 분석·차트 |
| 크리에이티브 | Figma, Canva | 브리프→광고 변형·상품컷 |
| 영업 | Salesforce, HubSpot | 우선 고객·미팅·CRM·클로즈 |
| 제품설계 | Figma, Canva | 와이어→프로토타입·URL 플로우 감사 |
| 주식 | FactSet, S&P, PitchBook | 실적·비교·논리 검증 |
| IB | 리서치·실사 | 피치·컴프 |

재무·PE·마케팅·컨설·법무 플러그인과 파트너 생태 예정. **SaaS 오케스트레이션 레이어**.

## Sites: 한 문장으로 공유 가능한 페이지

Business/Enterprise 프리뷰. 파일 대신 **웹 페이지** URL 공유.

- 재무: Excel→브라우저 시나리오 플래너
- 런치 허브: 카피·마일스톤·오너·결정
- 고객 리뷰 전: 업데이트·이슈·트렌드·다음 액션

몇 주 개발이 몇 분 대화로. 변경 시 Codex가 페이지 업데이트.

## Annotations: 선택 후 부분 수정

코드에서 **문서·스프레드시트·슬라이드**로 확장. 선택 영역만 편집.

## Codex와 ChatGPT: 수렴

채팅은 입구, 워크플로 실행이 본체. **ChatGPT** 와 한 팀에서 밀고 있습니다.
